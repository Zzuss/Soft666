# 数字化校园平台 - 技术设计文档

## 1. 系统架构

### 1.1 架构选型
本项目采用**轻量级Java Servlet/JSP Web应用**架构，符合课程要求。

**选型理由**：
- 满足架构限制要求（无Spring Boot）
- 支持Web界面，用户体验更好
- 便于部署和演示
- 技术栈成熟稳定

### 1.2 技术栈

| 层级 | 技术 | 版本/说明 |
|------|------|----------|
| 语言 | Java | Java 8+ |
| Web容器 | Servlet/JSP | Servlet 3.1, JSP 2.3 |
| 服务器 | Apache Tomcat | 9.x |
| 构建工具 | Maven | 3.6+ |
| 数据存储 | JSON/CSV/XML | 本地文件系统 |
| 前端 | HTML/CSS/JavaScript | 原生 +

## 2. 模块设计

### 2.1 模块划分

| 模块 | 职责 | 主要功能 |
|------|------|----------|
| 用户模块 | 用户管理 | 注册、登录、权限管理、个人信息 |
| 图书模块 | 图书管理 | 图书CRUD、查询、借阅管理 |
| 课程模块 | 课程管理 | 课程CRUD、选课、退课 |
| 成绩模块 | 成绩管理 | 成绩录入、查询、统计 |
| 校园卡模块 | 卡片管理 | 充值、消费、挂失 |

### 2.2 核心类设计

```
entity/
├── User.java              # 用户实体
├── Student.java           # 学生实体
├── Teacher.java           # 教师实体
├── Admin.java             # 管理员实体
├── Book.java              # 图书实体
├── BorrowRecord.java      # 借阅记录
├── Course.java            # 课程实体
├── Enrollment.java        # 选课记录
├── Grade.java             # 成绩实体
└── CampusCard.java        # 校园卡实体
```
